#  LogGuard-Analyzer

**LogGuard-Analyzer** is a lightweight, efficient Python automation tool that processes server log files to provide actionable insights and security monitoring.

##  Features
- **Traffic Analysis:** Extracts and counts unique IP addresses and their total request volume.
- **Security Monitoring:** Specifically filters for `HTTP 403 Forbidden` errors to identify unauthorized access attempts.
- **Automated Reporting:** Generates a structured `security_report.txt` file summarizing all suspicious activities.
- **Robust Error Handling:** Designed to handle missing files and empty log entries gracefully.

##  Technologies Used
- **Python 3.13**
- **File I/O & OS Module**
- **Dictionary-based Data Aggregation**
